Проект: aqara-hub-m2 — веб‑графики для Aqara Hub M2
====================================================

Этот проект даёт простой веб‑интерфейс с графиками показаний датчиков,
полученных от Aqara Hub M2 по локальной сети.

Технологии:
- Python 3.4 (Debian 8)
- Flask (легкий веб‑фреймворк)
- SQLite (хранение данных)
- Chart.js (красивые графики в браузере)

## Установка

Установи зависимости (на сервере Debian 8, в каталоге проекта):

```bash
cd /home/hontein/project/Aqara-Hub-M2
pip3 install -r requirements.txt
```

## Запуск

```bash
python3 app.py
```

После запуска открой в браузере:

- `http://IP_СЕРВЕРА:5000/` — страница с графиками.

## Дальнейшая доработка

- Реализовать сбор данных из Aqara Hub M2 и запись в базу.
- Добавить новые типы графиков.
